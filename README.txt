Проект представляет собой окно авторизации. В данном окне есть функции "войти" и "зарегистрироваться". При регистрации
логин и пароль вносятся в базу данных, при входе проверяется их наличие в той же базе. Если программа находит логин
в базе данных при регистрации, она выводит сообщение об ошибке, таккже как в случае, если она не находит логина при
входе.
